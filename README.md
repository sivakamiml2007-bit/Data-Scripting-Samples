
# Python Logic & Data Scripting Suite

This repository showcases foundational Python scripts developed to solve complex logic and data handling tasks.

### Key Implementations:
* **Data Normalization:** Scripts to standardize user input and handle string immutability (`indoor.py`, `playback.py`).
* **Mathematical Modeling:** Implementing physics formulas like E=mc² to handle large integer operations (`einstein.py`).
* **Data Sanitization:** Utilities to clean "dirty" financial data for precise arithmetic (`tip.py`).
* **Pattern Mapping:** Functional programming for text-to-emoji translation (`faces.py`).
